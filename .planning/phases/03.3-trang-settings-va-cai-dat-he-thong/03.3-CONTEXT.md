# Phase 03.3: Trang Settings và Cài đặt hệ thống - Context

**Gathered:** 2026-04-16
**Status:** Ready for planning

<domain>
## Phase Boundary

Xây dựng trang Settings tích hợp vào vùng **Admin** trên One-Map Isometric Canvas. Bao gồm:
1. Cài đặt cá nhân (Profile, Theme, Ngôn ngữ, Thông báo, Đổi mật khẩu)
2. **World Editor** — Chỉnh sửa vị trí, kích thước, góc xoay của các model 3D (Tủ sách, Giường, Lò rèn...) và kích thước/vị trí các vùng (zone) trên bản đồ isometric.

Không bao gồm: Quản trị hệ thống (Admin panel) — tách phase riêng sau.

</domain>

<decisions>
## Implementation Decisions

### Nội dung Settings
- **Nhóm settings:** Profile (tên, avatar) + Theme (dark/light) + Ngôn ngữ (Vi/En) + Thông báo + Đổi mật khẩu qua Supabase Auth
- **World Editor:** Chỉnh sửa các model 3D trong vùng — position (x, y), scale, rotation. Chỉnh kích thước và vị trí của từng zone.
- **Profile avatar:** Hỗ trợ cả chọn từ preset (animal icons, chibi faces) VÀ upload ảnh tự do (Supabase Storage)
- **Đổi mật khẩu:** Form đổi pass trực tiếp trong Settings qua Supabase Auth API

### Giao diện hiển thị
- **Panel 1 (Sidebar trái)** hiển thị danh sách nhóm Settings khi vào vùng Admin (giống Folder Tree ở vùng Docs)
- **World Editor** kết hợp: Click model trên map để chọn (hiện gizmo handles) + panel bên cạnh để chỉnh giá trị số chính xác (sliders/inputs)
- **Chỉnh zone** cũng kết hợp: kéo handles trực quan trên map + panel số để tinh chỉnh
- **Edit Mode toggle** — Nút "🔧 Chỉnh sửa không gian" nằm trong Panel 1 Settings. Khi bật: vùng đang chỉnh sáng lên + overlay mờ các vùng khác + hiện grid hỗ trợ căn chỉnh

### Luồng tương tác
- **Vào Settings:** Chỉ từ vùng Admin trên map — không có shortcut trên Header
- **Agent tương tác kết hợp:** Panel 1 hiện ngay khi vào vùng Admin. Nếu click vào object cụ thể trên sàn (gương → Profile, đèn → Theme...) thì Agent chạy đến + jump thẳng tới mục đó
- **Edit Mode nằm trong Panel 1:** Là 1 mục trong danh sách settings
- **Lưu thay đổi World Editor:** Preview + Confirm — Chỉnh thoải mái trong Edit Mode, khi tắt Edit Mode thì hỏi "Lưu thay đổi?" với option Lưu/Huỷ

### Lưu trữ cài đặt
- **Cài đặt cá nhân:** Supabase (bảng `user_settings`) — đồng bộ mọi thiết bị
- **World Config:** Supabase (bảng `world_config`) + giá trị mặc định. Có default layout sẵn, user chỉnh → lưu Supabase. Có thể reset về default
- **Cấu trúc World Config:** JSON đơn (1 record per user) — toàn bộ config là 1 JSON blob `{ zones: {...}, models: {...} }`. Dễ backup/restore/reset

### Claude's Discretion
- Thiết kế cụ thể gizmo handles (xoay/kéo/resize) cho model 3D trong không gian isometric
- Layout grid hỗ trợ khi Edit Mode bật
- Chọn preset avatars phù hợp với aesthetic Digital Atrium
- Mapping object → settings group trong vùng Admin (chọn icon/model phù hợp cho từng nhóm)

</decisions>

<specifics>
## Specific Ideas

- Agent click vào đồ vật trong vùng Admin để nhảy thẳng tới mục settings tương ứng (gương → Profile, đèn → Theme, búa/anvil → World Editor)
- World Editor gizmo trên isometric map: handles cho drag (di chuyển), corner handles cho resize, rotation arc cho xoay
- Edit Mode có overlay mờ + grid snapping để căn chỉnh chính xác
- Preview + Confirm workflow: chỉnh thoải mái → tắt Edit Mode → dialog "Lưu thay đổi?" → Apply hoặc Revert
- Default world config hardcoded trong code, user override lưu Supabase

</specifics>
