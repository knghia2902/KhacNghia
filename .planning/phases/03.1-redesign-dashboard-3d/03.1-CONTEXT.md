# Phase 03.1: Redesign Dashboard 3D - Context

**Gathered:** 2026-04-15
**Status:** Ready for planning
**Source:** Chat discussion

<domain>
## Phase Boundary

Redesign the `Home.jsx` (Dashboard) from a static 2D page into an interactive 3D Isometric "Digital Atrium" Hub.

</domain>

<decisions>
## Implementation Decisions

### 3D Core Layout
- Transform `Home.jsx` to use a full-screen `fixed inset-0` with isometric camera perspective.
- Create 4 navigation zones (Docs, Tools, Gallery, Admin) surrounding a central Home spawn point.

### Interactivity & Navigation
- The Chibi Agent will spawn at the center of the grid.
- Click-to-Walk: Clicking a zone sends the agent walking to those coordinates.
- Route triggers: Once the agent reaches a target coordinate, trigger React Router to navigate to the associated page.

### Visual Assets (Hologram Mode Approved)
- Instead of using external `.glb` models for the 4 zones, we will use **Pure CSS 3D Transforms** to construct glowing Hologram pedestals/zones. This guarantees instant 0-second loading and high performance.

</decisions>
