# Phase 03.1 Validation Strategy

This document supplements the Phase 03.1 plan by defining the architecture and criteria for validating the system post-implementation.

This phase is primarily a complex visual and interactive frontend update mapping to `Home.jsx`. 

## 1. High-Level Validation Goals

Ensure that:
1. The Home screen visually acts as a 3D isometric dashboard without rendering glitches.
2. Clicking on Hologram pedestals correctly moves the agent coordinates to the target.
3. Arriving at the target reliably triggers React Router (`navigate`) without double-clicking.
4. Smooth CSS transitions and proper walking state handling occur on interaction.

## 2. Test Architecture

*   **Integration Level:** Manual visual integration inspection. Testing click events and route transitions.
*   **Performance Consideration:** CSS Pedestals are used over GLB models to ensure 0-second TTI (Time to Interactive). Ensure `isometric.css` does not cause reflow bottlenecks.

## 3. Specific Component Test Plans

### A. Isometric Layout (`Home.jsx`)
*   **Viewport Constraints:** Ensure `fixed inset-0` centers the `isometric-world` completely natively on all resolutions.
*   **Z-Index Verification:** The Agent must be capable of moving "in front of" or "behind" pedestals based on standard DOM rendering bounds, although purely flat overlapping is acceptable for V1.

### B. Agent Pathfinding System (`Home.jsx`)
*   **State Verification:** Ensure the `onClick` event captures the exact coordinate of the Pedestal from `dataset` or local object array.
*   **Animation State Machine:** 
    *   State 0: Idle Animation (`src` toggled to walking = false)
    *   State 1: Walking Animation (`src` = walking) + CSS `translate` + `setTimeout(navigate)`
*   **Routing Logic Check:** Validate that `navigate('/docs')` happens precisely at the end of the CSS transition duration.

## 4. End-to-End Walkthrough

1. Go to Home (`/`).
2. Verify the 4 pedestals exist (Docs, Tools, Admin, Gallery).
3. Click "Docs".
4. Character turns, replaces model URL to `Running/Walking`, slides towards Docs over 2 seconds.
5. React Router automatically loads the Docs Dashboard page seamlessly.
