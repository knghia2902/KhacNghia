# Technology Stack
- **Frontend Framework**: React 19
- **Build Tool**: Vite
- **Language**: JavaScript (ES Modules, no TypeScript)
- **Styling**: Tailwind CSS v4, PostCSS
- **Routing**: React Router DOM (v7)

# Key Libraries
- **Rich Text / Media**: `@tiptap` suite for rich text editing, `@uiw/react-md-editor` for markdown editing, `react-image-crop`.
- **Drag and Drop**: `@dnd-kit/core`, `@dnd-kit/sortable`, `@dnd-kit/utilities`
- **Document Processing**: `file-saver`, `html-docx-js`, `mammoth` (for DOCX parsing).
- **Markdown Handling**: `react-markdown`, `remark-gfm`, `rehype-highlight`, `rehype-sanitize`.
